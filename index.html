automatic system.new.dat extractor

Requiremnets

  1. Python v3.x
  2. 7-Zip V16.x

Note: make sure your environment variables path is correctly set for python and 7-Zip (set it in system properties-->advance-->environment variable-->path)

How To (convert_dat.cmd)

1) set environmental variable path for the files

2) extract your rom.zip into a folder

3) now within the extracted folder open cmd prompt and type "convert_dat"

4) it will automatically detect the files and extract into a folder named system

5) have fun tweaking

How To (convert_dat2.cmd)

1) extract system_new_dat_extractor.zip into a folder and open convert_dat2.cmd

2) select your rom zip file for which you want to extract system.new.dat

3) wait for few seconds

4) folder named system will open, once it has finished extracting

5) have fun tweaking
